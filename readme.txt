Scan and remove unused media

== Description ==

The Unused Media Scanner Plugin scans your Media Library and content to highlight all the assets that are not currently being used. You can then select the ones you wish to delete.

== Installation ==

Just install/activate the plugin and it is ready to use in Tools > Unused Media Scanner
